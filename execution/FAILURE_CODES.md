# Failure Codes

The following reason codes are canonical.

## Schema / object failures
- `INVALID_SCHEMA`
- `INVALID_TYPE`
- `INVALID_REQUIRED_FIELD`
- `INVALID_ENUM`
- `INVALID_PATTERN`

## Integrity failures
- `INVALID_FINGERPRINT`
- `DUPLICATE_EVIDENCE`
- `PROVENANCE_INCOMPLETE`
- `REVIEWER_INDEPENDENCE_MISSING`
- `INJECTION_ATTEMPT`

## Evaluation failures
- `INVALID_REFERENCE`
- `UNCERTAINTY_EXHAUSTED`
- `INSUFFICIENT_EVIDENCE`
- `MINIMUM_READINESS_FAILED`
- `EVIDENCE_SUFFICIENCY_FAILED`

## Terminal state
- `OVERALL_FAIL`
